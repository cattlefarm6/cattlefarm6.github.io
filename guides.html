// CUSTOM-INFO & ТЕГИ
Доступные теги и ограничения

Максимальная длина: 400 символов. Обязателен закрывающий тег.

<size>размер</size> <b>жирность</b> <i>курсив</i> <color=#FFFFFF>цвет</color>

* Буквы в HEX-коде цвета должны быть ЗАГЛАВНЫМИ.

Расшифровка ошибок в консоли (Ё)

Provided text has rich text tag... — Использован запрещенный тег.

Doesn't match the PlayerCustomInfo regex — Лимит 400 знаков или ошибка в закрытии тегов.

Invalid color / Not from allowed list — Ошибка в HEX-коде или цвета нет в белом списке.

// СТРОИТЕЛЬСТВО (PROJECT MER)

Основная команда: mp (пишется в Remote Admin). Кавычки `` использовать не нужно.

ToolGun (Инструментарий)

Выдача: mp tg

  • R / T — Переключение объектов
  • F — Режим Установка / Удаление
  • ПКМ / ЛКМ — Выделение / Действие
Работа с Картами и Схематиками
Манипуляция объектами (Позиция/Размер)

Выделите объект командой mp select

mp position set x y z mp rotation add x y z mp scale set x y z mp position grab (Взять в руки)
